Subtitle Edit

Link de descarga - Shot changes - Atajos de teclado - Funciones

En esta sección, encontrarás una lista de recursos y herramientas que te ayudarán a maximizar tu experiencia con Subtitle Edit.

Subtitle Edit es un editor de subtítulos gratuito y de código abierto para Windows que te permite crear, editar y sincronizar subtítulos de una manera muy fácil y rápida.

Atajos de teclado

  • Tecla play/pausa: Ctrl + P
  • Ajustar tiempo inicial: Ctrl + W
  • Ajustar tiempo final: Ctrl + E
  • Insertar un nuevo subtítulo en la posición del video Shift + F9
  • Bajar a la segunda línea del subtítulo: Enter
  • Avanzar un cuadro: Ctrl + →
  • Retroceder un cuadro: Ctrl + ←
  • Avanzar 500 ms: Alt + →
  • Retroceder 500 ms: Alt + ←
  • Ir al siguiente subtítulo: Alt + ↓
  • Ir al subtítulo anteriror: Alt + ↑
  • Cursiva: Ctrl + i
  • Buscar o remplazar palabras/frases/etc.: Ctrl + F
  • Símbolo de los tres puntos: Alt + 0133

Funciones

  • Seleccionar y editar varios subtítulos a la vez: Haz clic sobre cada subtítulo mientras presionas Shift. Esto te ahorrará mucho tiempo cuando tengas que cambiar el posicionamiento de varios subtítulos, ponerlos en cursiva, etc. Ya no tendrás que editar uno por uno.
  • Cambiar la posición de los subtítlos: Clic derecho sobre el subtítulo > Alignment
  • Editar atajos de teclado: Options > Settings > Shortcuts
  • Cambiar Time Code Mode a frames: Options > Settings > Time code mode... HH:MM:SS:FF
  • Exportar texto sin timecodes para pasar el corrector: Archivo > Exportar > Exportar texto plano...
  • Comparar traducción/transcripción con otra: Archivo > Comparar...
  • Corregir errores comunes: Herramientas > Errores comunes...
